Collection Panel




Collection panel contains all node collections you own. Node collection are set of nodes that you can reuse in any projects. You can also share node collection with others through Steam Workshop or file sharing.


The folder on the left reflects the actual directory structure at the %DIRR%/Collections folder.



Menus



Settings



Colection settings dialog allow you to modify how the collections are displayed.


Refresh


Refresh the collection, used when modifying the collection from outside.


Open in file explorer


Open the current opening folder in the native file explorer.


Add folder


Create new folder at the current location.


Add selecting node as collection


Create a new collection with the nodes selected in the graph panel.


View


Cycle view settings between grid and list.



Collection Management


Creating Collection


To create a collection, select nodes in the graph panel, then click on the "Add selecting node as collection" button.



A collection need to be inside a folder. Make sure you enter a folder first or else the button will be disabled.



You can modify the metadata of the collection by clicking on the hamburger menu button.



Context Menu



Right clicking on a collection will brings you a context menu. Here you can do several things.


To update a collection, select the new collection, and select "Replace with selected".
