Gradient Selector




Gradient selector allow you to modify a gradient.



Gradient Manipulation


Add/Remove Keys


Left click on the gradient to add key at the mouse position. Right click while hovering on the key to remove it.


Change Key Blending


Click on "Key Blending" on the top right to choose how the graedient blend colors between each key.



Preset


Presets allow for reusing gradient in other nodes, projects. Gradient preset is stored in the %DIRR%/Gradients folder.


Apply Gradients


Left click on the preset to set it as the current gradient.


Add to Preset


Click on "Add to preset" to add current gradient to preset.


Delete Preset


Right click on the preset and select "Delete gradient" to delete it permanently.
