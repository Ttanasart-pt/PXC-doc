Color adjust node is use for color grading.



Properties


Brightness


Adjust the brightness of the image by adding brightness value uniformly across the surface.


Exposure


Adjust the exposure of the image by multiplying the exposure value to the pixel. This make the dark pixel stays dark and the bright pixel becomes brighter.


Contrast


Adjust the contrast of the image, the different between darker and brighter colors.


Hue


Change the hue of the image.


Saturation


Adjust the saturation of the image.


Value


Adjust the value of the image.



Blend


Blending properties apply Blend color to a pixel based on the Blend Mode and Blend Amount.

The detail on blend mode can see in Blend.
