Pixel Sort

Sort pixel by brightness in horizontal

Pixel sorting is an effect where pixels in the same horizontal or vertical axis are being sorted by the brightness. This node is recommended to be animate using the feedback loop.




Properties



The number of sorting iteration.



The sorting group threshold. Incerasing the amount will separate the sorting into multiple segments.
