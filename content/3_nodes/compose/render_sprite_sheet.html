Render Spritesheet

Create spritesheet from image array or animation.



Node Data

Display name

Render Spritesheet

Internal name

Node_Render_Sprite_Sheet

Inheritances

node
node_render_sprite_sheet

Inputs

Outputs

Render Spritesheet generate a single surface containing frames from animation or array. It is recommended that every frames should have the same size.



Input


There are 2 types of input set by the : Animation and sprite array.

When set to animation, the node will require full rendering to record all the animation frames. You can also use to skip frames for smaller spritesheet.

When set to sprite array, the node will runs automatically similiar to Image Grid.



Output Properties



Control how the surface is arranged. Horizontal and Vertical mode will place the surfaces linearly. Grid will place image in a grid.



When the input surfaces has different sizes, control where to place the smaller surface:




Set the space between each surface. If the is set to grid, then this value will becomes vec2 for horizontal and vertical spacing.



Add padding space in each direction afterward.
