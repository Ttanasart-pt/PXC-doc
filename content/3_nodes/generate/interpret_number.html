Interpret Number convert array of numbers into surface by map the value into a range.

This node can be useful for creating animation by animating the number array instead of the surface itself.




Properties


Number


The array of number to interpret.


Mode


Select intepretation modes:


  • Greyscale: Interpret value into greyscale color by mapping to the Range scale.
  • Gradient: Takes the value from the range mapping and use it to sample color from the Gradient.

Range


The range to map the value in. Min value will correspond to 0 and Max value will correspond to 1.


Gradient


The gradient to map to if use the Gradient mode.
