Flood fill node fill a region with the same color with a new color. That region is based on a single points. If you want to map all regions with new colors, you can use <node region_fill> node.

<img-deco flood_fill/>

## Properties

### <junc position>
Position of a pixel in the region to fill.

### <junc threshold>
Threshold of the color difference to ber consider the same region.

### <junc colors>
Color to fill to.

### <junc diagonal>
Whether to fill diagonal pixel.

### <junc blend>
Blending mode between the original and the filled color.
