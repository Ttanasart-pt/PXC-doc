MK Tile generate tileset from a set of surfaces.



Layers


Tile set generation has 3 layers, the background , tile , and edges . The surface provided to every layers should be tilable/seamless texture.




Tile Set


The control the set of tiles to be generated. Currently there're 3 sets:


GMS Corner


Tile set for GameMaker studio 17 sprites tile.




GMS Corner + Side


Tile set for GameMaker studio 55 sprites tile.




Godot Blob


Tile set for Godot tile.




Cropping


The property control the trimming of the tile layer in each direction. Note that the crop only affect tile layer.




Edge


Edge layer is used to draw sprite on the edge of the tile. Note that edge



Edge type control whether to use a single surface for all edges or individual edge sprites.



When using Uniform type, the edge should be aligned to the top edge with the same dimension as the tile. The other edges will be generated by flipping or rotating the edge texture based on the value.




When using individual type, each edge should be aligned to the side it connected to.



Edge sprite property allows you to set the corner edge differently from the side.




Shift the edge of each side.



Whether to crop the edge of use the entire sprite.




Output


control whether to output as a single spritesheet or sprite array.
