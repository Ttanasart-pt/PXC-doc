Generate triangular grid pattern.




Properties

Rendering Modes


Colored Tile


Fill each cell with a solid random color sampled from the value.



Height Map


Render each cell out as a heightmap.



Texture Grid


Render out the surface value from to each cell.



Texture Sample


Fill each cell with a solid color sampled from the value.
