This node will search throught every pixel in the image and returns the position of the pixel with a given color. With [0, 0] be the top left of the image, x axis goes to the right and y axis goes down.



Properties


Property Description
Color of the pixel to find
How much the color can differ from the search color to be match
Returns all matches pixel instead of the first pixel
Include alpha channel in the search
How much the alpha channel can differ from the search color to be match
