Audio Loudness

Calculate volume of an audio bit array.



Node Data

Display name

Audio Loudness

Internal name

Node_Audio_Loudness

Inheritances

node
node_processor
node_audio_loudness

Inputs

Outputs

This node takes a array of audio data and calculate loudness of that segment in decibel.



Loudness Calculation


Loudness is calculated using the following formula:


$$L = 10 \cdot \log_{10}\left(\frac{1}{N} \sum_{n=0}^{N-1} a[n]^2\right)$$


Where:

  • L is the loudness in decibel
  • N is the number of samples
  • a[n] is the audio data
