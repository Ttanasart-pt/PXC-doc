WAV File In node allows you to import .wav audio file as a audioBit data. The node also read the file attributes such as sample rate, no. of channels, and duration.


To output as a single channel, choose the Mono option.


The Sync Length button allows you to set the animation length to the same as the audio duration.



Previewing


The node will playback the audio when playing the animation. The inspector action is used to toggle the playback. Most attributes are used to control the playback:


Property Description
Gain Playback volume
Shift Shift playback time to sync with the timeline
