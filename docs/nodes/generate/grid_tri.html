Triangle Grid

Generate triangular grid pattern.



Node Data

Display name

Triangle Grid

Internal name

Node_Grid_Tri

Inheritances

node
node_processor
node_grid_tri

Inputs

Outputs

Generate triangular grid pattern.




Properties



The thickness of the gap between cell.



Rendering Modes


Colored Tile


Fill each cell with a solid random color sampled from the value.



Height Map


Render each cell out as a heightmap.



Texture Grid


Render out the surface value from to each cell.



Texture Sample


Fill each cell with a solid color sampled from the value.



Truchet Properties


Similiar to other grid related nodes. Truchet settings rotate the surface randomly to create an interesting effect.



The ratio of the rotation.



Apply random rotation on top of the truchet.
