Remap UV map using plane.

The process of mapping a 2D image to a 3D model is call UV mapping. This node allow you to remap the UV by projecting texture onto the model in world space.




Subobject


For object with multiple textures, each textrue will be assigned to a subobject. This property allow you to target a specific subobject to remap the UV insted of all.



Bake


Click the "Bake" button to freeze the current mapping. Without this option, all the transformation prior to this node will be applied to the UV mapping.
