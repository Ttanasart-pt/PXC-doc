Introduction

1.18.0




Pixel Composer is a node based VFX compositor for pixel art. Using nodes to pass data and apply different effects. It also support keyframe-based animation with timeline.


This documentation aims to help you with basic functionalities of Pixel Composer. From its user interface to the several nodes it offers.


Pixel Composer is in active development and this documentation will be update as the software evolves.
